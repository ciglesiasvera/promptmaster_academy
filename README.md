# PromptMaster Academy - Plataforma de Ingeniería de Prompts

## Resumen Ejecutivo
PromptMaster Academy es una plataforma web educativa diseñada para guiar a usuarios, desde novatos hasta expertos, en la creación metódica de prompts de alta calidad para Inteligencia Artificial. A través de un enfoque paso a paso, la herramienta asegura la aplicación de técnicas avanzadas como *Few-shot Learning* y *Chain of Thought*.

Este proyecto es parte de una actividad práctica del curso **IA Lab – Smart Work (b2c-ia-lab-agosto-2025) 2.0 *** de **SkillNest**.

## Características
* **Constructor Guiado:** Flujo lógico (Rol -> Contexto -> Formato).
* **Soporte Multi-Dominio:** Adaptable a Leyes, Salud, IT, etc.
* **Persistencia Híbrida:** LocalStorage para invitados, MySQL para usuarios registrados.
* **Mobile First:** Diseño responsivo con Bootstrap 5.

## Despliegue
1. Clonar repositorio.
2. Importar `database.sql` en MySQL.
3. Configurar `db.php`.
4. Ejecutar en servidor Apache/Nginx.
